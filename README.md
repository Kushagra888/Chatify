## Setup .env file

```js
PORT=5000
MONGO_DB_URI=mongodb://localhost:27017/chatify
JWT_SECRET=your_jwt_secret_key
NODE_ENV=development
```

## Installation

1. Clone the repository
2. Install dependencies:
   ```shell
   npm install
   ```
3. Start the development server:
   ```shell
   npm run dev
   ```

## Build the app

```shell
npm run build
```

## Start the app

```shell
npm start
```
